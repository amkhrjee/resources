Programming Languages

JavaScript

Although there are many YT tutorials out there, I'd highly recommened learning JS from javascript.info. It's like the missing documentation to the language.

Python

The official tutorial is good enough. Head First Python is good too.

C/C++

Although learning C is not a prerequisite to learning C++, I recommened you learn C first anyway. The folllowing are excellent books for getting started with C.
  • - The C Programming Language
  • - Effective C: An Introduction to Professional C Programming
  • - Expert C Programming: Deep Secrets
For C++, learncpp.com provides excellent tutorials.

Rust

Although I am no expert in Rust, I learnt it (and still learn, when needed) from Brown University's fork of the official Rust book.

Computer Architecture

I think these two books work as excellent introduction to the subject:
